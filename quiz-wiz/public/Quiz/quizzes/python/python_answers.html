Python Programming Quiz - Correct Answers

1. What is the result of 5 + 3 * 2 in Python?

Correct Answer: 11

Explanation: Python follows the order of operations (PEMDAS). The multiplication is performed first, so 3 * 2 = 6, then 5 + 6 = 11.

2. Which of the following functions is used to read input from the user in Python?

Correct Answer: input()

Explanation: The input() function is used in Python to read input from the user as a string.

3. Which data type is used to store whole numbers in Python?

Correct Answer: int

Explanation: int is the data type used in Python to store whole numbers (integers), such as 1, -50, or 1000.

4. Which of the following is a valid Python variable name?

Correct Answer: variable_1

Explanation: In Python, variable names must start with a letter or an underscore and can be followed by letters, numbers, or underscores. 1variable is not valid because it starts with a number.

5. Which of the following is used to define a function in Python?

Correct Answer: def

Explanation: Functions are defined using the def keyword followed by the function name and parentheses in Python.

6. What is the output of the following code: print(3 + 4 * 2)?

Correct Answer: 11

Explanation: The multiplication is performed first, so 4 * 2 = 8, then 3 + 8 = 11.

7. Which of the following is used to create a list in Python?

Correct Answer: []

Explanation: Lists in Python are created using square brackets, [], with elements separated by commas, e.g., [1, 2, 3].

8. What does the following Python code do: len("Python")

Correct Answer: Counts characters

Explanation: The len() function returns the number of characters in a string. In this case, it returns 6 because "Python" has 6 characters.

9. Which Python data type is ordered and changeable, and allows duplicate members?

Correct Answer: list

Explanation: list in Python is ordered, mutable, and allows duplicates. Lists are created using square brackets, e.g., [1, 2, 2, 3].

10. Which of the following is the correct syntax to create a dictionary in Python?

Correct Answer: {}

Explanation: Dictionaries in Python are created using curly braces {}, with key-value pairs separated by colons, e.g., {"name": "John", "age": 30}.
