HTML Quiz - Answers

Below are the correct answers and explanations for each question.

Question 1:

Correct Answer: Hyper Text Markup Language

Explanation: HTML stands for Hyper Text Markup Language, which is the standard language for creating web pages.

Question 2:

Correct Answer: <a>

Explanation: The <a> tag is used to create hyperlinks, which are used to navigate between web pages.

Question 3:

Correct Answer: <img>

Explanation: The <img> tag is used to embed images in HTML documents.

Question 4:

Correct Answer: <ol>

Explanation: The <ol> tag defines an ordered (numbered) list.

Question 5:

Correct Answer: <tr>

Explanation: The <tr> tag is used to define rows in a table.

Question 6:

Correct Answer: alt

Explanation: The "alt" attribute provides alternative text for an image if the image cannot be displayed.

Question 7:

Correct Answer: <title>

Explanation: The <title> tag defines the title of the document shown in the browser's title bar.

Question 8:

Correct Answer: <br>

Explanation: The <br> tag inserts a single line break in the text.

Question 9:

Correct Answer: <select>

Explanation: The <select> tag is used to create a drop-down list.

Question 10:

Correct Answer: <meta>

Explanation: The <meta> tag provides metadata about the HTML document, such as author or keywords.
