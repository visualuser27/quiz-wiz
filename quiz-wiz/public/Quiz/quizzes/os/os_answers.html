Operating System Quiz - Correct Answers

1. What is the function of an operating system?

Correct Answer: All of the above

Explanation: An operating system manages system resources, controls hardware, and performs various essential tasks to make the computer function properly.

2. Which of the following is an example of an operating system?

Correct Answer: Linux

Explanation: Linux is a widely used open-source operating system. Python, Java, and MySQL are programming languages or databases, not operating systems.

3. Which of the following is a type of operating system process?

Correct Answer: Both of the above

Explanation: Both user processes and kernel processes are types of processes managed by an operating system.

4. What is virtual memory in an operating system?

Correct Answer: Memory management technique

Explanation: Virtual memory is a memory management technique that provides an "idealized abstraction" of the storage resources, which creates the illusion to users of a very large (main) memory.

5. Which of the following is NOT a file system type?

Correct Answer: HDFS

Explanation: NTFS, FAT32, and RAMFS are file system types, while HDFS (Hadoop Distributed File System) is a distributed file system used in big data processing.

6. What does the term 'multitasking' refer to in an operating system?

Correct Answer: Executing multiple processes at once

Explanation: Multitasking refers to the ability of an operating system to execute multiple tasks or processes concurrently, either by switching between them rapidly (time-sharing) or running them truly concurrently on multiple processors.

7. Which of the following is the main function of the operating system's kernel?

Correct Answer: All of the above

Explanation: The kernel is responsible for managing processes, memory, devices, and providing an interface between the software and hardware of the system.

8. What is a system call in an operating system?

Correct Answer: A request to the kernel to perform a task

Explanation: A system call is a mechanism that allows user-level processes to request services from the operating system kernel, such as I/O operations or process management.

9. What does the 'deadlock' problem refer to in operating systems?

Correct Answer: When processes cannot access required resources

Explanation: Deadlock occurs when two or more processes are blocked forever because they each hold a resource and are waiting for the other process to release a resource, creating a circular dependency.

10. Which of the following is used to create a process in an operating system?

Correct Answer: fork()

Explanation: The fork() system call in Unix-like operating systems is used to create a new process by duplicating the calling process.
