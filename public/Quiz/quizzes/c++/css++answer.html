C++ Programming Quiz - Correct Answers

1. What is the difference between `++i` and `i++` in C++?

Correct Answer: a) `++i` increments `i` before using it, whereas `i++` increments `i` after using it.

Explanation: `++i` increments the value of `i` before it is used, while `i++` increments the value after using it. This is important when the operator is part of an expression.

2. Which of the following is used to allocate memory dynamically in C++?

Correct Answer: b) `new`

Explanation: In C++, the `new` keyword is used for dynamically allocating memory. `malloc()` and `calloc()` are used in C, not C++.

3. What is the purpose of the `virtual` keyword in C++?

Correct Answer: b) It allows a function to be overridden in a derived class.

Explanation: The `virtual` keyword in C++ ensures that the correct function is called for an object, even if the object is accessed via a pointer to a base class.

4. What does the `const` keyword do when applied to a pointer in C++?

Correct Answer: c) Both the pointer and the object it points to cannot be modified.

Explanation: When the `const` keyword is applied to a pointer, it means the pointer cannot be reassigned to point to another object, and the object it points to cannot be modified.

5. Which C++ feature is used to allow a class to use properties and methods from another class?

Correct Answer: c) Inheritance

Explanation: Inheritance allows one class to inherit the properties and behaviors of another class, enabling code reuse and extension.

6. What is the correct syntax to create a reference variable in C++?

Correct Answer: a) `int &a = 10;`

Explanation: A reference variable in C++ is declared with the `&` symbol, and it must be initialized with a valid object or value.

7. What will be the output of the following C++ code?


            int x = 5;
            std::cout << x << ++x << x++ << std::endl;

Correct Answer: b) `5676`

Explanation: The operators `++x` and `x++` have different effects on `x`. `++x` increments the value before using it, while `x++` increments it after using it. The order of evaluation impacts the output.

8. Which of the following is used to define a constant in C++?

Correct Answer: b) `const`

Explanation: The `const` keyword is used to declare constants, which cannot be changed after initialization in C++.

9. What is the default value of a static variable in C++?

Correct Answer: a) Zero

Explanation: In C++, static variables are initialized to zero by default if no other value is provided.

10. Which of the following C++ containers allows for fast random access and insertion/deletion at the end?

Correct Answer: a) `vector`

Explanation: `vector` in C++ provides fast random access and efficient insertion and deletion at the end. It is one of the most commonly used container classes.
