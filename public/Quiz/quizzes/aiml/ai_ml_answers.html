AI/ML Quiz: Correct Answers and Explanations

1. What does the term "Machine Learning" refer to?

Answer: Data-driven learning

Explanation: Machine learning refers to algorithms that learn from data to improve their performance without explicit programming. It primarily relies on data to make predictions or decisions.

2. Which of the following is a supervised learning algorithm?

Answer: Linear Regression

Explanation: Linear Regression is a supervised learning algorithm that attempts to model the relationship between a dependent variable and one or more independent variables using a linear equation.

3. What is the main idea behind "Supervised Learning"?

Answer: Learning from labeled data

Explanation: Supervised learning involves learning a function that maps input data to the correct output, based on labeled training data. The algorithm is "supervised" by these labels during training.

4. Which library is commonly used for deep learning in Python?

Answer: TensorFlow

Explanation: TensorFlow is an open-source library for numerical computation that is widely used for deep learning applications. It provides tools for building and training neural networks.

5. Which of the following is an example of unsupervised learning?

Answer: K-means Clustering

Explanation: K-means is an unsupervised learning algorithm used for clustering, where data is grouped based on similarities without labeled outputs. It creates 'K' clusters based on the data features.

6. What does the acronym "AI" stand for?

Answer: Artificial Intelligence

Explanation: AI refers to Artificial Intelligence, the simulation of human intelligence in machines that are programmed to think and act like humans, including learning, reasoning, and problem-solving.

7. What is the purpose of "Cross-validation" in machine learning?

Answer: To evaluate model performance on unseen data

Explanation: Cross-validation is a technique for assessing how well a machine learning model will generalize to new, unseen data. It involves partitioning the dataset into multiple subsets and training/testing the model on these subsets.

8. Which type of neural network is commonly used for image classification?

Answer: Convolutional Neural Network (CNN)

Explanation: CNNs are specialized deep learning architectures designed to process data with grid-like topology, such as images. They use convolutional layers to automatically extract features from images for classification tasks.

9. Which of the following is a common evaluation metric for classification problems?

Answer: Accuracy

Explanation: Accuracy is the ratio of the number of correct predictions to the total number of predictions made. It is widely used to measure the performance of classification models.

10. Which type of machine learning algorithm is used for spam detection?

Answer: Supervised Learning

Explanation: Spam detection is a classification problem where emails are labeled as "spam" or "not spam." Supervised learning algorithms, such as decision trees and support vector machines, are commonly used for this type of task.
