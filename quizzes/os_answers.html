Operating Systems Quiz - Answers

Below are the correct answers and explanations for each question.

Question 1:

Correct Answer: Windows

Explanation: Windows is a widely used operating system. MS Word and Chrome are applications, while Google is a company.

Question 2:

Correct Answer: A running program

Explanation: A process in an operating system refers to an active execution of a program, including its code and state.

Question 3:

Correct Answer: Linux

Explanation: Linux is an open-source operating system, widely used for servers, and is known for its security and flexibility.

Question 4:

Correct Answer: Kernel

Explanation: The kernel is the core part of an operating system, managing hardware resources and enabling communication between software and hardware.

Question 5:

Correct Answer: Android

Explanation: Android is an operating system primarily used in mobile devices, based on the Linux kernel.

Question 6:

Correct Answer: Virtual Memory

Explanation: Virtual memory is a memory management technique that gives an "idealized abstraction" of the storage resources that are actually available on a given machine.

Question 7:

Correct Answer: File System

Explanation: A file system is a way of storing and organizing computer files and data.

Question 8:

Correct Answer: Multitasking

Explanation: Multitasking refers to the ability of an operating system to execute more than one task simultaneously.

Question 9:

Correct Answer: System Call

Explanation: A system call provides the interface between a process and the operating system.

Question 10:

Correct Answer: BIOS

Explanation: BIOS (Basic Input/Output System) is the firmware that initializes hardware during the booting process of a computer.
