C++ Quiz - Answer Key

1. Who is the creator of C++?

Correct Answer: Bjarne Stroustrup

Explanation: C++ was created by Bjarne Stroustrup in 1979 at Bell Labs as an extension of the C programming language.

2. Which of the following is a valid C++ identifier?

Correct Answer: _variable

Explanation: In C++, identifiers can start with an underscore, but cannot start with a number or use reserved keywords.

3. What is the correct syntax to output "Hello, World!" in C++?

Correct Answer: cout << "Hello, World!"

Explanation: In C++, the cout object is used to print text to the console, and the text should be enclosed in double quotes.

4. What is a constructor in C++?

Correct Answer: A function called automatically

Explanation: A constructor is a special member function in a class that is called automatically when an object of that class is created. Its purpose is to initialize the object.

5. Which of the following is not a fundamental data type in C++?

Correct Answer: string

Explanation: In C++, string is not a primitive data type. It is a class from the C++ Standard Library, whereas int, char, and float are primitive types.

6. What does the "new" keyword do in C++?

Correct Answer: Allocates memory dynamically

Explanation: The new keyword is used to allocate memory on the heap during runtime. It returns a pointer to the allocated memory.

7. What is the file extension for C++ files?

Correct Answer: .cpp

Explanation: C++ source code files typically use the .cpp extension. Other extensions like .cc or .cxx are also used in some environments.

8. Which of the following is used to declare a constant in C++?

Correct Answer: const

Explanation: The const keyword is used in C++ to define a constant value that cannot be changed after its initial assignment.

9. What does the "this" pointer represent in C++?

Correct Answer: The current object

Explanation: The this pointer in C++ refers to the current instance of the object for non-static member functions.

10. What is polymorphism in C++?

Correct Answer: Ability to take multiple forms

Explanation: Polymorphism allows objects of different types to be treated as objects of a common base type, primarily through inheritance and function overloading/overriding.
